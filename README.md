# Tech Store Website

This project is a Tech Store website developed for a university course.  
It is built using HTML, CSS, and JavaScript, and it uses localStorage as a simple database.

## Features

- Product listing with images and descriptions
- Add to cart functionality
- Dynamic cart with subtotal, tax, and total calculation
- Order system with customer information
- Review system
- Light and Dark mode toggle
- Local storage used as database

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage

## Project Structure


tech-store-website/
- index.html
- style.css
- script.js
- images/
- products/


## How to Run

1. Download or clone the repository  
2. Open `index.html` in your browser
